Introducción

Contexto y justificación del proyecto

Hoy en día, muchas personas necesitan un sistema operativo seguro, estable y económico, pero no siempre conocen todas las opciones disponibles. Windows es el más usado, pero su costo legal y problemas de privacidad hacen que algunos usuarios busquen alternativas. MacOS también queda descartado por su precio y percepción de complejidad.

En este contexto, Linux se presenta como una opción ideal: es gratuito, seguro, personalizable y respaldado por una gran comunidad. Sin embargo, todavía existe falta de información accesible para usuarios principiantes, lo que dificulta su adopción.

Por ello, nos hemos propuesto crear esta web que acerque Linux a cualquier persona interesada, ofreciendo información clara y recursos prácticos. Así, se impulsa el uso del software libre y se ayuda a que más usuarios puedan trabajar con un sistema seguro sin gastar dinero.

Con el aumento de la cuota de mercado de Linux y la popularidad de distribuciones como Linux Mint, Ubuntu o Fedora, esta iniciativa responde a una tendencia actual y contribuye al crecimiento del software libre en la sociedad.

Objetivos del proyecto

  • Explicar qué es Linux y sus beneficios respecto a seguridad, coste, privacidad y personalización.
  • Guiar al usuario en la elección de una distribución adecuada según su nivel y necesidades.
  • Proporcionar tutoriales básicos para la instalación, configuración y uso de Linux.
  • Dar a conocer el software libre y fomentar su uso responsable y consciente.
  • Contribuir a la difusión de Linux entre usuarios principiantes y curiosos.
  • Diseñar una web accesible y fácil de navegar, pensada tanto para usuarios novatos como para quienes quieran mejorar sus conocimientos.

Alcance y Limitaciones del proyecto

El proyecto comprende la creación de una página web informativa sobre Linux, diseñada principalmente para usuarios principiantes y usuarios interesados en migrar desde Windows.

La web incluirá contenido sobre ventajas, distribuciones populares, instalación, herramientas básicas y seguridad en Linux.

Se trabajará con información verificada y accesible, fomentando el uso del software libre.

Conclusión y análisis del mercado

El desarrollo de este proyecto ha permitido analizar y comprender la importancia del software libre y, en particular, del sistema operativo Linux como una alternativa real y accesible frente a los sistemas comerciales.

El análisis del sector ha mostrado que Linux ocupa una posición sólida en el ámbito empresarial y tecnológico, y que su crecimiento en entornos domésticos y educativos representa una oportunidad de expansión para el software libre. Asimismo, el proyecto contribuye a varios Objetivos de Desarrollo Sostenible (ODS), promoviendo la educación de calidad, la innovación y el consumo responsable mediante la reutilización de equipos informáticos.

Metodología del proyecto

Para la realización de este proyecto, se pasó por múltiples etapas antes de crear el resultado final:

  • Investigación y análisis: Observamos que con la reciente descontinuación del soporte de Windows 10, miles de ordenadores se quedarían obsoletos contribuyendo a la obsolescencia programada y generando residuos electrónicos. Investigando, descubrimos Linux como solución de código abierto, compatible y accesible para cualquier usuario.
  • Diseño del proyecto y planificación: Seleccionamos un estilo y estructura atractivos para los usuarios, con información concisa y clara. Usamos HTML, CSS y algo de JavaScript.
  • Desarrollo web: La web está dividida en secciones: inicio, información, historia, guías, foro y formulario de contacto.
  • Pruebas y validaciones: La web se presentó a varias personas de distintos rangos de edad para evaluar estética, distribución y comprensión. Las conclusiones fueron muy positivas.
